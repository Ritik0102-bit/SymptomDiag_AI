# 🩺 SymptomDiag | AI Clinical Assistant

**SymptomDiag** is an interactive, artificial intelligence-powered symptom checker designed with a sleek, dark-mode clinical terminal interface. Users can input their symptoms, and the AI engine processes the data to provide an initial differential analysis and suggest potential common conditions.

🚨 **DISCLAIMER:** *This application was developed strictly as a student project for INT 428 (AI Essentials). It is for educational and demonstrative purposes only. SymptomDiag is NOT a substitute for professional medical advice, diagnosis, or treatment. Always seek the advice of your physician or other qualified health provider with any questions you may have regarding a medical condition.*

## ✨ Features
* **AI-Powered Analysis:** Integrates the Google Gemini 1.5 Flash Large Language Model to interpret user symptoms contextually.
* **SecOps/Clinical UI:** A custom-designed, responsive dark-mode interface that mimics an advanced medical diagnostic terminal.
* **Serverless Architecture:** Runs entirely in the browser using Vanilla JavaScript, making it lightning-fast and easily deployable via GitHub Pages.
* **Contextual Memory:** The chatbot retains the history of the current session to provide follow-up questions and accurate analysis.

## 🛠️ Tech Stack
* **Frontend:** Semantic HTML5, CSS3, Vanilla JavaScript (ES6+)
* **Styling:** Tailwind CSS (via CDN) & Custom Glassmorphism CSS
* **Icons:** Google Material Symbols
* **AI Engine:** Google Generative AI (Gemini API)
